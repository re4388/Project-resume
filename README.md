## Python toy program

place_gui_v2.py

    調用Google Map API，自動搜尋住址附近，包裝為exe檔方便執行。
    用法：點擊exe檔案 => 輸入想要查詢字串(譬如，小吃店) => Enter

auto-account.py

    記帳系統，點兩下執行檔，輸入金額和項目，自動存入預定的雲端excel檔案。
    用法：點擊exe檔案 => 輸入金額 => Enter => 輸入項目 => Enter
